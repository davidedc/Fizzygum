    - (void) blit(aCanvas, clippingRectangle = @bounds)

    This method only paints this very morph's "image", it doesn't descend the children recursively. The recursion mechanism is done by recursivelyBlit, which eventually invokes blit. Note that this morph might paint something on the screen even if it's not a "leaf".
